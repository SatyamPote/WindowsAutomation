## What's Changed

- First public release of **Lotus.app** — a native macOS menu-bar control panel for Mac-MCP.
- Telegram bot integration: chat with your Mac from anywhere, gated by allowed user IDs.
- First-run setup wizard: Telegram token, allowed users, name, and Ollama model picker.
- LaunchAgent integration — the bot service auto-starts at login and survives crashes.
- HTTP control API on `localhost:40510` for status, logs, restart, and stop.
- MCP server (`mac-mcp`) exposing macOS desktop tools (keyboard, mouse, screenshots, filesystem) to Claude Desktop / Claude Code.
- Drag-to-install DMG with a custom lotus-pond banner background.

## Install

1. Download `Lotus-1.0.0.dmg` below and drag **Lotus.app** to **Applications**.
2. Inside the `Mac-MCP/` folder, run `uv sync` to populate the Python venv.
3. Launch Lotus, point it at the `Mac-MCP/` folder, enter your Telegram token + Ollama model, and hit **Save & Launch Bot**.

> The DMG is **ad-hoc signed** (not notarized). On first launch, right-click → **Open**, or run:
> ```bash
> xattr -d com.apple.quarantine /Applications/Lotus.app
> ```

See [SETUP.md](https://github.com/SatyamPote/Lotus/blob/main/Mac-MCP/SETUP.md) for the full guide.
