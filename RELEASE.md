# 🌸 Lotus Windows AI Control Agent v2.2.0-STABLE 🌸

We are proud to announce the first stable release of **Lotus**, the ultimate AI-powered Windows control agent. This version focuses on **architectural stability**, **reliability**, and **professional-grade interaction**.

## 🚀 What's New in v2.2.0

### ⚖️ Hardened Command Routing
The heart of Lotus has been rebuilt. We've implemented a **Strict Priority Routing System** that ensures mission-critical commands bypass the AI entirely for 100% reliability.
- **Priority**: System > Files > Music > Research > AI.
- **Direct File Access**: Commands like `open <file>` and `send <file>` now work instantly by searching your storage and download folders.

### 🔍 Enhanced Research Intelligence
The Research Engine is now more powerful than ever, featuring a multi-source fallback system.
- **Fail-Safe Search**: Uses Wikipedia as primary, falling back to DuckDuckGo and Web Scraping.
- **Professional Reports**: Automatically generates high-quality PDF reports with summaries, key findings, and references.
- **Resilient Image Scraping**: Improved image scraper with relative URL resolution ensures you get visual context for every topic.

### 🎵 Minimal Stable Music System (v2.2.0)
We've removed all complex TUI and playlist logic to prioritize **100% reliability**.
- **Instant Playback**: Direct integration with `mpv` and YouTube search.
- **Aggressive Cleanup**: Intelligent `taskkill` ensures zero ghost processes or overlapping audio.
- **Simplified Controls**: Focus on `play`, `stop`, and `pause` for a crash-free experience.
- **Single Response UI**: Clean, framed Telegram replies with no bot chatter.

### 🗣️ Sweet Voice & Standardized UI
- **Female Voice Feedback**: All assistant responses are delivered in a sweet, professional female tone.
- **Premium Framing**: Every bot response is wrapped in a clean, professional frame for a superior user experience.
- **Spam-Free Interaction**: Optimized logic ensures exactly one clean response per command.

## 🛠️ Installation

1. Download `LotusSetup.exe` from the assets below.
2. Run the installer and follow the prompts.
3. Add your Telegram Bot Token and your Allowed User ID.
4. Restart Lotus from the desktop shortcut.
